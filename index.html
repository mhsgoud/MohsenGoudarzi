Ongoing projects

  • pdeDir.com
    pdeDir.com is a side project which I dedicate time to to explore new ideas, enhance my programming skills and simultanesoly addressing a computationally intensive problem. The ultimate goal of this project is to have a web-based general purpose application for numerical solving of partial differential equations. The idea is to have a cloud based solution for addressing dedicated similations that the end user runs on a server. The kernel library (written in C++) is the outcome of my PhD and postdoc works. For the server side, I use node.JS and C++, and JavaScript functionalities are integrated to the C++ kernel by using the power of node JS. Parallel comupting and CUDA are used to speed up calculations of the Kernel, e.g. for assembly of stiffness matrices, iterative solvers and post-processing. For the front-end and 3D visualization, I use JQuery and visualization libraries such as vtk.js and threejs.
